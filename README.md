# Advent of Code 2021

Solutions to [Advent of Code in year 2021](https://adventofcode.com/2021).

To run a solution, install [Kotlin](https://kotlinlang.org) first. On macOS:

```shell
brew install kotlin
```

then use `run.sh`, e.g.:

```shell
./run.sh 01
```
